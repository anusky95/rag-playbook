## 🚀 Recommended Learning Path

### New to RAG?
Start here → `0_foundations` → `1_embeddings` → `2_retrieval` → `4_generation` → `13_tutorials_workshops`

### Already know the basics?
Jump to `3_search_techniques` → `5_advanced_rag` → `6_evaluation` → `7_deployment`

### Domain-specific use cases?
See `8_case_studies` and `9_industry_verticals`

### Want the latest techniques?
Go to `10_cutting_edge_research` and `11_tools_frameworks`
